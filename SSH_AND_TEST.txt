TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

If something fails:

  1. Check Docker:
     docker-compose -f docker-compose.demo.yml ps

  2. Restart backend:
     docker-compose -f docker-compose.demo.yml restart backend

  3. Check logs:
     docker-compose -f docker-compose.demo.yml logs --tail=50 backend

  4. Check database:
     docker exec ai-dialer-postgres psql -U postgres -d ai_dialer -c "SELECT COUNT(*) FROM calls;"
